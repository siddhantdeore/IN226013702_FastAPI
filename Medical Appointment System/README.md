# Medical Appointment System - FastAPI

## 📌 Project Description
This project is a backend system built using FastAPI to manage doctors and appointments.

## features
- GET APIs
- POST APIs with validation
- CRUD operations
- Appointment workflow
- Search, sorting, pagination

## Technologies Used
- Python
- FastAPI
- Uvicorn

## Run the Project
pip install fastapi uvicorn
uvicorn main:app --reload

## API Testing
http://127.0.0.1:8000/docs
